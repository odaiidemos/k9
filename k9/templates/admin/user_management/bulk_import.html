كيفية استخدام الاستيراد الجماعي
  1. قم بتحميل ملف القالب (Excel أو CSV) باستخدام الزر أدناه
  2. افتح الملف وأدخل بيانات المستخدمين حسب الأعمدة المحددة
  3. احفظ الملف وارفعه باستخدام النموذج أدناه
  4. سيتم عرض نتائج الاستيراد مع أي أخطاء إن وجدت

name الاسم الكامل للمستخدم نعم
username اسم المستخدم (حروف إنجليزية وأرقام فقط) نعم

  • سيتم توليد كلمات مرور آمنة تلقائياً للمستخدمين
  • تأكد من عدم تكرار أسماء المستخدمين أو البريد الإلكتروني
  • الملفات المدعومة: Excel (.xlsx, .xls) و CSV (.csv)
